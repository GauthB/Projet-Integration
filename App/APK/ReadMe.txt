Pour installer l'application sur votre android : 
- Transférez le fichier (PeopleFlux.apk) dans les fichiers dans votre smartphone.
- Dans les paramètres de sécurité  de votre téléphone activez les sources inconnues.
- Retrouver votre fichier apk et clickez dessus, (une confirmation vous sera peut-être demandée, cliquez sur "installer quand même").
- l'installation devrait se lancer.
